Setup Guide

How to set up TaskFlow

Get your task manager live on the internet for free in about 10 minutes. No coding experience needed.

01

Download the file

Click the download button at the top of this page to save taskflow.html to your computer. This single file is your entire app — no extra files needed.

💡 Tip: Save it somewhere easy to find, like your Desktop.
02

Create a free GitHub account

Go to github.com and sign up for a free account. GitHub is where your app will be hosted — completely free, forever.

https://github.com/signup
💡 Tip: Pick a username you like — it will appear in your app's web address.
03

Create a new repository

Once logged in, click the + icon in the top right and select New repository. Name it taskflow, set it to Public, then click Create repository.

💡 Tip: It must be Public for free hosting to work.
04

Upload your file

Inside your new repository, click Add file → Upload files. Drag in your taskflow.html file. Before saving, rename it to index.html — this is important. Then click Commit changes.

taskflow.html → rename to → index.html
05

Turn on GitHub Pages

Go to your repository's Settings tab, then click Pages in the left sidebar. Under Source, select Deploy from a branch, choose main, and click Save.

💡 Tip: It takes about 1–2 minutes for your site to go live after saving.
06

Your app is live!

After a minute or two, visit your app at the address below — replacing yourusername with your GitHub username.

https://yourusername.github.io/taskflow
💡 Tip: Share this link with anyone — it works on mobile too.
07

Want to accept real payments?

To charge users for Student, Personal, or Business plans, sign up for a free Stripe account at stripe.com. Create payment links for each plan and replace the plan buttons in the code with your Stripe links. No coding knowledge needed — Stripe has a visual link builder.

https://stripe.com → Products → Payment Links
💡 Tip: Stripe is free to set up. They take a small % per transaction (around 2.9% + $0.30).
